# Transduction Gui

Write a GUI in pyqt6.

## Inputs

Input boxes (QDoubleSpinBox) for with the value being the default value:
Cavity Parameters
gopt = 0.025
gmm = 0.091
kappaOpt = 10.0
kappaMM = 10.0

Atom Parameters
Natoms = 5e5
AtomRadius = 250e-6
gammaOpt = 6.0
gammaMM = 0.100

Beam Parameters
WaistUV = 1.2e-3
WaistBlue = 1.2e-3

## Outputs

Follow the example in test.py and generate a plot in the gui.
Every change to the input boxes should force a recalculation of the plot.
